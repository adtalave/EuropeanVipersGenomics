# Unveiling the evolutionary history of European vipers and their venoms from a multi-omic approach

Repository from the manuscript "Unveiling the evolutionary history of European vipers and their venoms from a multi-omic approach".


Here you will find scripts for:

Filter reads, mapping reads to a reference genome, SNP-calling, VCF filtering.
PCA
Calculating [genome-wide heterozygosity estimates](https://github.com/adtalave/EuropeanVipersGenomics/blob/main/heterozygosity_estimates.sh) in previously delimited [windows](https://github.com/adtalave/EuropeanVipersGenomics/blob/main/windows.sh).
